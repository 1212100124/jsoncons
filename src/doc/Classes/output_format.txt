    jsoncons::output_format

The output_format class is an instantiation of the basic_output_format class template that uses char as the character type.

Floating point serialization attempts to produce digits in decimal format. If it cannot, it produces digits in exponential format. By default trailing zeros are removed, except the one immediately after a decimal point. The period character (‘.’) is always used as the decimal point, non English locales are ignored.  A `precision` gives the maximum number of significant digits, the default precision is `16`.
