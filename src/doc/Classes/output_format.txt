    jsoncons::output_format

The output_format class is an instantiation of the basic_output_format class template that uses char as the character type.

### Header

    #include "jsoncons/output_format.hpp"

### Member constants

    least_trailing_zeros
    fixed
    scientific
Floating point formats. The default format is `least_trailing_zeros`

    default_indent
The default indent is 4

### Constructors

    output_format()

    output_format(bool indenting)

### Accessors

    bool indenting() const
Returns `true` if indenting is turned on, the default is `false`.

    size_t indent() const
Returns the level of indentation, the default is 4

    size_t precision() const 
Returns the floating-point decimal precision, the default is 16 for the `least_trailing_zeros` floating point format, and 17 for `fixed` and `scientific`.

    bool remove_trailing_zeros() const
